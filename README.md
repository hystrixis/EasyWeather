# 易用天气（EasyWeather）
易用天气是一款基于Android端开源的天气预报软件，能够查看不同城市的天气数据。易用天气中的天气数据由和风天气提供。代码遵循Apache v2 License开源协议。本软件主要作为学习和交流使用。
